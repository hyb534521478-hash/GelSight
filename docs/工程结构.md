# 工程结构

```text
gelsight_force_calibration_project/
├── config/default.yaml                  # 路径、对齐阈值、训练参数
├── requirements.txt                     # Python依赖
├── install_to_desktop.sh                # 复制工程到Ubuntu桌面目标目录
├── scripts/
│   ├── 00_check_dataset.py              # 检查109个片段和txt是否对应
│   ├── 01_preprocess_align.py           # 时间对齐预处理
│   ├── 02_split_dataset.py              # 8:2训练/验证划分
│   ├── 03_train_method1_img0_imgn.py    # 方法1训练
│   ├── 04_train_method2_optical_flow.py # 方法2训练
│   ├── 05_train_method3_temporal.py     # 方法3训练，窗口3/5/7
│   ├── 06_train_method4_ref_temporal.py # 方法4训练，窗口3/5/7
│   ├── 07_evaluate_checkpoint.py        # 模型复评
│   ├── 08_realtime_run.py               # 实时推理，r复位Img0
│   ├── run_all_train.sh                 # 一键跑全部训练
│   └── setup_conda_env.sh               # 创建conda环境
├── src/gelsight_force_calib/
│   ├── preprocess.py                    # 接触点检测和时间对齐核心逻辑
│   ├── split.py                         # 片段级8:2划分
│   ├── datasets.py                      # 四种训练输入的Dataset
│   ├── models.py                        # ResNet18和MLP模型
│   ├── train.py                         # 统一训练循环
│   ├── evaluate.py                      # checkpoint评估
│   ├── realtime.py                      # OpenCV实时软件
│   ├── flow_features.py                 # 光流点阵位移特征
│   ├── transforms.py                    # 图像预处理
│   ├── io_utils.py                      # 文件名解析、txt读取、插值
│   └── config.py                        # 配置读取
└── docs/
    ├── GPT55_Codex_调试流程.md
    └── 工程结构.md
```
